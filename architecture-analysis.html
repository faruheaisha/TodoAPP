▸ 推荐方案：Tauri v2 + React + TypeScript

核心理由：Windows 11 内置 WebView2 运行时（即 Edge 的内核），Tauri 直接复用它而不像 Electron 需打包整个 Chromium。这意味着你的 app 安装包只有 ~3MB（vs Electron 150MB+），内存占用极低（~20MB vs 100MB+），对于一个小型待办工具而言，这是决定性优势。

Anthropic 风格 UI：使用 Web 技术（React + Tailwind CSS）可以完美还原 Anthropic 官网的现代设计风格，这是 WPF/XAML 几乎无法做到的，Flutter 也需要额外工作。

系统级功能：Tauri v2 官方提供 autostart、notification、tray、store 等插件，一站式覆盖开机自启、托盘图标、Windows Toast 通知、本地持久化等需求，无需第三方库。

    participant Win as Windows OS
    participant Auto as Autostart Plugin
    participant App as TodoApp
    participant Timer as Delay Timer
5min后触发
    participant Notify as Notification Plugin
    participant User as User

    Win->>Auto: 系统开机/登录
    Auto->>App: 启动 TodoApp (后台)
    App->>Timer: 注册延迟任务 (5分钟)
    App->>App: 系统托盘图标就绪

    Timer->>Notify: 时间到达
    Notify->>User: Toast弹窗 "是否打开待办助手？"
    User->>Notify: 点击 "开启"
    Notify->>App: 打开主窗口

    App->>App: 加载待办数据
    App->>App: 检查截止日期提醒

    loop 每次开机后检查
        App->>Notify: 发现即将截止的任务
        Notify->>User: 弹窗提醒 "任务即将到期！"
    end

    Note over App,User: 用户也可通过托盘图标随时打开
